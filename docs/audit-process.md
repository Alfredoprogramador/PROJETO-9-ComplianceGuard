# Processo de auditoria contínua

1. Coletar evidências de políticas, execuções e aprovações.
2. Assinar evidências e registrar hashes para integridade.
3. Consolidar trilha de auditoria por controle e regulação.
4. Gerar relatórios para auditoria interna/externa.
5. Abrir plano de ação para desvios detectados.
